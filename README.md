# FOODY_ZONE

#Overview
* Foody Zone is a responsive food website built with React.js, designed to provide an intuitive and mobile-friendly experience for users looking to explore various meal options.

#Features
* Responsive Design: The website adapts seamlessly to different screen sizes, ensuring a great user experience on both desktop and mobile devices.
* Mobile-friendly interface: Optimized for touch interactions and smaller screens, making navigation easy on smartphones and tablets.
* Search functionality: Quickly find food items using a search box.
* Meal category buttons: Easily browse options like Breakfast, Lunch, and Dinner with dedicated buttons.


#Technologies Used
* React.js : For building the user interface
* CSS : For styling and responsive design
